# Use Cases (High-Level)

This list captures the major system use cases across departments. It is scope-complete for planning, but may evolve as detailed workflows are refined.

## Identity & Access
- Create organization
- Manage organization details
- Create property
- Manage property details
- Create department
- Manage department assignments
- Create user
- Invite user
- Suspend/reactivate user
- Assign roles to user
- Remove roles from user
- Assign user to department(s)
- Set primary department
- Define roles
- Define permissions
- Map permissions to roles

## Service Orders / Ticketing
- Create service order
- Update service order details
- Assign service order
- Reassign service order
- Start service order
- Pause/stop service order
- Defer service order
- Complete service order
- Cancel/void service order
- Add remarks/notes
- Attach photos/documents
- Record parts/labor costs
- Search/filter service orders

## Inspections
- Define inspection template/checklist
- Schedule inspection
- Perform inspection (pass/fail/N/A)
- Score inspection
- Submit inspection report
- Review inspection history

## Housekeeping
- Generate daily housekeeping tasks
- Assign rooms to housekeepers
- Update room cleaning status
- Verify/inspect cleaned rooms
- Track housekeeping performance

## Maintenance
- Create corrective maintenance task
- Schedule preventive maintenance
- Generate PM tasks from schedule
- Track maintenance logs
- Record asset/equipment details
- Scan QR to open asset task

## Guest Experience
- Log guest complaint/issue
- Notify responsible team
- Track complaint resolution
- Follow up with guest
- Analyze guest satisfaction trends

## Guest Communication (Guest-Facing Requests)
- Submit guest request (web/app)
- Route request to department
- Update request status
- Notify guest of completion

## Risk & Compliance
- Create compliance checklist
- Record audit/inspection findings
- Track corrective actions
- Review compliance dashboards
- Store legal/contract records

## Projects
- Create project
- Create snagging list
- Track technical audit items
- Close project tasks

## Food & Beverage
- Track breakfast counts
- Assign outlet tasks
- Monitor outlet readiness

## Corporate
- Manage suppliers
- Manage contracts
- Track purchase orders
- Track CAPEX approvals

## Energy & Sustainability
- Record energy KPIs
- Analyze energy trends
- Generate efficiency reports

## Reporting & Analytics
- Dashboard KPIs (by department)
- Export reports
- Schedule reports

## Audit & Tracking
- Write audit log entry
- View audit history
- View entity change history
